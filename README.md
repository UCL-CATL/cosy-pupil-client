cosy-pupil-client
=================

A Matlab client of [cosy-pupil-server](https://github.com/UCL-CATL/cosy-pupil-server).

Works on both GNU/Linux and Windows.

Dependencies
------------

- [cosy-zeromq-matlab](https://github.com/UCL-CATL/cosy-zeromq-matlab) >= 1.0
